## Golden Rules Quick Reference

These principles override other rules (Rule 1.2):

1. **Card text beats rules** - If a card contradicts a game rule, the card text supersedes that rule
2. **Prohibition beats permission** - Effects that prevent something override effects that allow it
3. **Do as much as you can** - Resolve as much of an effect as possible even if some parts can't be done
4. **Choices during resolution** - Choices for effects are made during resolution, not when playing a card

## Common Mechanics

### The Bag (Rule 7.7)

Triggered abilities wait in "the bag" before resolving:

- Only triggered abilities are added to the bag
- Active player chooses which ability to resolve next
- When a player has no more triggered abilities in the bag, it passes to the next player in turn order
- Each ability resolves one at a time

### Game State Check (Rule 1.8)

Happens after every action/ability resolves:

- Check win conditions (20+ lore wins)
- Check loss conditions (tried to draw from empty deck)
- Check damage (characters/locations with damage ≥ willpower are banished)
- Then triggered abilities from these events go into the bag

### Multiplayer (Rule 9)

- Turn order proceeds to the left
- Active player acts first for simultaneous effects
- When a player loses, all their cards/effects leave the game
- Simultaneous turns for team play
