# Deck Building - By Topic Index

**Comprehensive Rules Version 2.0.1** - Effective February 5, 2026

## Overview

This topic covers the rules for building legal Disney Lorcana decks, including deck size, ink types, and card limits.

## Deck Construction Rules

**Rule refs:** 1.10, 2.1

### Core Constructed Rules

| Requirement     | Rule                               |
| --------------- | ---------------------------------- |
| **Deck Size**   | 60 or more cards (no upper limit)  |
| **Ink Types**   | Up to 2 different ink types        |
| **Card Limits** | Up to 4 cards with same full name  |
| **Card Types**  | Standard Disney Lorcana cards only |

### Standard Cards

- Standard Disney Lorcana cards are 2.5" x 3.5"
- Have Illuminarry icon on back
- Non-standard cards (different back, oversized) not allowed

---

## Ink Types

**Rule refs:** Card classification line

### What Are Ink Types?

- Represented by symbols on cards (Amber, Amethyst, Emerald, Ruby, Sapphire, Steel)
- Each card has 1 ink type (or none)
- Deck can have up to 2 different ink types

### Building with 2 Ink Types

- Can include cards of both ink types
- Can include cards with no ink type (if any exist)
- Can't use 3rd ink type even if "just a few cards"

### Inkless Decks

- Cards without ink type count toward ink type limit
- Can build deck with 0, 1, or 2 ink types
- No advantage to using fewer ink types

## Card Limits

**Rule refs:** 1.10

### Full Name

- Card's name + version
- Example: "Mickey Mouse - Brave Little Tailor" = full name
- Example: "Mickey Mouse - True Friend" = different full name

### 4-Copy Limit

- Can have up to 4 cards with same full name
- Applies to EACH full name separately
- Can have 4 "Mickey - Brave Little Tailor" AND 4 "Mickey - True Friend"

### Card Type Examples

**Characters:**

- 4x "Mickey Mouse - Brave Little Tailor"
- 4x "Mickey Mouse - True Friend"
  These are different full names - both allowed!

**Actions/Items/Locations:**

- Same name = same full name (no version on these card types)
- Can only have 4 of same action/item/location

---

## Card Types for Deck Building

### Characters (5.3)

- Have classifications (Hero, Villain, etc.)
- Have Ⓒ and ⣇
- Can have multiple versions of same name

### Actions (5.4)

- One-time effects
- Resolve then discard
- Same name = same full name

### Items (5.5)

- Ongoing effects or abilities
- Same name = same full name

### Locations (5.6)

- Provide ongoing benefits
- Can have multiple copies with same name in deck
- Name + version = full name

### Songs

- Subtype of Action
- Can be sung using Singer/Sing Together keywords

## Limited Format (Pack Rush)

**Rule refs:** 10.2

Pack Rush uses different deck construction rules:

| Rule              | Normal              | Pack Rush                            |
| ----------------- | ------------------- | ------------------------------------ |
| **Deck Source**   | Constructed deck    | 2 booster packs                      |
| **Deck Size**     | 60+                 | No minimum (all cards from boosters) |
| **Starting Hand** | 7 cards             | 5 cards                              |
| **Starting Ink**  | 0 cards             | 2 marketing cards                    |
| **Win Condition** | 20 lore             | 15 lore                              |
| **Empty Deck**    | Lose at end of turn | Shuffle discard as new deck          |

### Pack Rush Setup

1. Take 2 unopened boosters
2. Remove marketing cards
3. Shuffle cards together
4. Put 2 marketing cards into inkwell
5. Draw 5-card opening hand
6. Play!

## Common Questions

**Q: Can I mix different ink types in my deck?**
A: Yes, up to 2 different ink types. Can also include inkless cards.

**Q: Do inkless cards count toward my ink type limit?**
A: Yes, cards without ink symbols still count as an ink type for deck building.

**Q: Can I have more than 4 of a character if they have different versions?**
A: Yes! Each "Name - Version" is a different full name. Can have 4 of each version.

**Q: What about actions, items, and locations?**
A: These don't have versions. Same name = same full name, max 4 copies.

**Q: Are there banned cards?**
A: Check the latest set release notes. As of February 2026, consult official sources for current ban list.

**Q: Can I use cards from any set?**
A: Yes, all Disney Lorcana cards are legal in Constructed. Check for any set-specific rules in release notes.

**Q: What's the minimum deck size?**
A: 60 cards for Constructed formats. No minimum for Pack Rush (use all cards from 2 boosters).
