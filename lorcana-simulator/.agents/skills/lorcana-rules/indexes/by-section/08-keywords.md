# Section 8: Keywords

**Comprehensive Rules Version 2.0.1** - Rule References 8.1-8.15

## Overview

This section covers all keyword abilities in Disney Lorcana. Keywords are abilities represented by short names that provide specific effects. Keywords with +[N] stack with other effects that provide that keyword.

---

## 8.1 General

**Rule refs:** 8.1.1-8.1.2

**Key Concepts:**

- Keywords are abilities with short, bolded names
- Normally have reminder text in italics (not rules text)
- Some keywords have +[N] - these stack with other effects

### Stacking Keywords (8.1.2)

- Keywords with +[N] stack with other effects
- Keywords without +[N] don't stack
- If card has non-stacking keyword and gets it again, only first applies

---

## 8.2 Alert

**Rule refs:** 8.2.1-8.2.2

**Keyword:**

- **Alert** = "This character ignores the challenging restriction of Evasive."

### Key Points:

- Static ability
- Character doesn't gain Evasive
- Can still gain Evasive from other effects
- Allows challenging Evasive characters

---

## 8.3 Bodyguard

**Rule refs:** 8.3.1-8.3.3

**Keyword:**

- **Bodyguard** = Two abilities:
  1. "This character may enter play exerted rather than ready."
  2. "If an opponent would choose one of your characters to challenge, they must choose this character or another character with Bodyguard if able."

### Key Points:

- First ability: Static, functions while being played
- Second ability: Static, creates challenging restriction
- Multiple Bodyguards = opponent chooses among them
- Doesn't apply if no Bodyguard character can be challenged

---

## 8.4 Boost

**Rule refs:** 8.4.1-8.4.3

**Keyword:**

- **Boost N** = "Once during your turn, you may pay N to put top card of your deck facedown under this card."

### Key Points:

- Activated ability with duration (your turn)
- Facedown card can't be looked at by any player
- Card under another isn't in play
- Doesn't count as playing that card
- Can use once per turn per source

---

## 8.5 Challenger

**Rule refs:** 8.5.1-8.5.2

**Keyword:**

- **Challenger +N** = "While this character is challenging, they gain +N Ⓒ."

### Key Points:

- Static ability
- Functions only while challenging
- Stacks (+N ability)
- Doesn't apply when being challenged
- Only adds to Ⓒ, not ⣇

---

## 8.6 Evasive

**Rule refs:** 8.6.1

**Keyword:**

- **Evasive** = "This [Type] can't be challenged except by a character with Evasive."

### Key Points:

- Static ability
- Creates challenging restriction
- Alert ignores this restriction
- Applies to all characters unless specified (e.g., "Evasive character" vs "Evasive Hero")

---

## 8.7 Reckless

**Rule refs:** 8.7.1-8.7.4

**Keyword:**

- **Reckless** = Two abilities:
  1. "This character can't quest."
  2. "You can't declare the end of your turn if this character is ready and can challenge an opposing character or location."

### Key Points:

- Both abilities are static
- Can't quest at all (even if dry)
- Can't end turn if ready and valid challenge target exists
- Can still exert to use abilities or sing songs

---

## 8.8 Resist

**Rule refs:** 8.8.1-8.8.3

**Keyword:**

- **Resist +N** = "Damage that would be dealt to this character or location is reduced by N."

### Key Points:

- Static ability
- Damage reduction (not damage counter removal)
- Stacks (+N ability)
- If reduced to 0, no damage considered dealt
- Doesn't affect damage counters put/moved onto card

---

## 8.9 Rush

**Rule refs:** 8.9.1

**Keyword:**

- **Rush** = "This character can challenge as though they were in play at the beginning of your turn."

### Key Points:

- Static ability
- Allows challenging while drying
- Character still drying (can't quest or pay costs)
- Only allows challenging, not other actions

---

## 8.10 Shift

**Rule refs:** 8.10.1-8.10.8.2

**Keyword:**

- **Shift** = "If you have a character in play with the same name as this card, you may play this character by paying their Shift cost instead of their ink cost. If you do, put this card on top of another character you have in play with the same name."

### Key Points:

- Static ability (alternate cost)
- Called "shifting" when used
- Must have same-name character in play
- Put shifted card on top of same-name character

### Entering Play (8.10.2-8.10.4)

- **On exerted character:** Enters exerted
- **On ready character:** Enters ready (dry if character was dry)
- **On drying character:** Enters drying
- Retains damage of character underneath

### Using Shifted Character (8.10.5)

- Can use all turn actions of character underneath
- Doesn't gain text of character underneath
- Keeps effects that applied to underneath character
- Can exert to sing if underneath was dry when entered

### Leaving Play (8.10.7)

- All cards in stack move to same zone
- No longer considered in stack

### Variants (8.10.8.1-8.10.8.2)

- **[Classification] Shift:** Shift onto character with specified classification
- **Universal Shift:** Shift onto any character

---

## 8.11 Singer

**Rule refs:** 8.11.1-8.11.2

**Keyword:**

- **Singer N** = "This character can pay the alternate cost of a song as though they had cost N instead of their normal ink cost."

### Key Points:

- Static ability
- Allows singing songs by exerting character
- Character counts as cost N only for singing
- Character's ink cost doesn't change
- Only during process of playing a card

---

## 8.12 Sing Together

**Rule refs:** 8.12.1-8.12.3

**Keyword:**

- **Sing Together N** = "Instead of paying the ink cost of this card, you can exert a number of your or your teammates' characters with total ink cost N or greater to play this card without paying its ink cost."

### Key Points:

- Static ability (alternate cost)
- Can use your OR teammates' characters
- Combine costs of multiple ready characters
- Total must meet or exceed N
- "Whenever this character sings" abilities trigger

---

## 8.13 Support

**Rule refs:** 8.13.1

**Keyword:**

- **Support** = "Whenever this character quests, you may add this character's Ⓒ to another chosen character's Ⓒ this turn."

### Key Points:

- Triggered ability
- Optional choice when questing
- Adds Ⓒ to another character
- Lasts until end of turn
- Multiple Supports can stack

---

## 8.14 Vanish

**Rule refs:** 8.14.1-8.14.2

**Keyword:**

- **Vanish** = "When this character is chosen by an opponent as part of resolving an action's effect, banish this character."

### Key Points:

- Triggered ability
- Triggers when chosen by opponent's action effect
- Resolves after action's effect resolves
- If character moved zones before resolves, does nothing

---

## 8.15 Ward

**Rule refs:** 8.15.1-8.15.2

**Keyword:**

- **Ward** = "Your opponents can't choose this card when resolving an effect."

### Key Points:

- Static ability
- Creates targeting restriction
- Doesn't prevent effects that don't choose
- Opponent can still affect with non-targeted effects

---

| **Boost +N**        | Activated | Put top card under this (once/turn)                    | Yes (+N) |

| **Reckless**        | Static    | Can't quest; can't end turn if ready and can challenge | No       |

| **Vanish**          | Triggered | Banish when chosen by opponent's action                | No       |
| **Ward**            | Static    | Opponents can't choose this card                       | No       |

## Keyword Interactions

**Common Interactions:**

- **Alert + Evasive:** Alert ignores Evasive restriction
- **Bodyguard + Evasive:** Can challenge Evasive if have Bodyguard
- **Bodyguard + Ward:** Ward can still be chosen if only Bodyguard available
- **Challenger + Resist:** Modified Ⓒ still reduced by Resist
- **Rush + drying:** Rush allows challenging while drying
- **Shift + Rush:** Shifted character can challenge if underneath was dry
