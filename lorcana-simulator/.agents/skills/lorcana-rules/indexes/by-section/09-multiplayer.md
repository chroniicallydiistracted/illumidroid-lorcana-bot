# Section 9: Multiplayer

**Comprehensive Rules Version 2.0.1** - Rule References 9.1-9.3

## Overview

This section covers additional rules for games with three or more players, including turn order, simultaneous effects, players leaving the game, and simultaneous turns for team play.

---

## 9.1 General

**Rule refs:** 9.1

**Key Concepts:**

- Multiplayer games have 3+ players
- Team games are subset not covered in this document
- Follow all two-player rules plus these additional rules

---

## 9.2 Multiplayer Games

**Rule refs:** 9.2.1-9.2.3

### 9.2.1 Turn Order

- At end of each player's turn, play proceeds to player on their left
- Continues clockwise around table
- Not back-and-forth like two-player games

### 9.2.2 Simultaneous Actions

When ability requires multiple players to act at same time:

- Active player goes first
- Proceed to left, one player at a time
- Continue until all players done

### 9.2.3 Player Leaves Game

When a player loses from ending turn with empty deck:

- Player must leave game immediately
- All their cards removed from game
- All their effects removed from game
- Triggered abilities in bag from them cease to exist
- Generated static effects with duration continue until duration ends

**Example:** Pete - Games Referee prevents playing actions. If Pete's player loses, Pete is removed but "can't play actions" effect continues until duration ends.

---

## 9.3 Simultaneous Turns

**Rule refs:** 9.3.1-9.3.2

Some multiplayer formats use simultaneous turns for teams.

### 9.3.1 During Simultaneous Turn

- All team members progress through phases at same time
- During Main Phase, can take any turn action in any player order
- One turn action must complete before next begins
- During End-of-Turn Phase, all declare end together
- All triggers added to bag at same time
- All "this turn" effects end at same time

### 9.3.2 Bag in Simultaneous Turns

- Triggered abilities added by multiple players added by team as whole
- Team chooses order to resolve
- Otherwise follows normal bag rules (Section 7.7)

**Example:** Three teammates taking simultaneous turn:

- Ready step: All ready cards at same time
- Set step: All gain location lore, triggers added to bag together
- Draw step: All draw at same time
- Main Phase: Each completes one action before next player acts
- End-of-Turn: All declare end together, triggers added together

## Quick Reference

| Rule                     | Multiplayer Difference                                                   |
| ------------------------ | ------------------------------------------------------------------------ |
| **Turn order**           | Proceeds to left (clockwise)                                             |
| **Simultaneous actions** | Active player first, then left                                           |
| **Player loses**         | Cards/effects removed immediately, static effects with duration continue |
| **Simultaneous turns**   | Team progresses together, chooses order for bag resolution               |
