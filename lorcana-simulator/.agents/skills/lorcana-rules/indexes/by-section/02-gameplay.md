# Section 2: Gameplay

**Comprehensive Rules Version 2.0.1** - Rule References 2.1-2.4

## Overview

This section covers the three stages of a Disney Lorcana game: Setup Stage, In-Game Stage, and Cleanup Stage.

---

## 2.1 General

**Rule refs:** 2.1.1-2.1.3

**Key Concepts:**

- Games are played in three stages: Setup, In-Game, Cleanup
- Turn order continues clockwise (to the left)
- When players must act simultaneously, active player goes first

### Determining Starting Player (2.1.1)

- Randomly determine before game begins
- Can use any mutually agreeable method

### Turn Order (2.1.2-2.1.3)

- Proceeds clockwise (to the left of active player)
- Active player acts first in simultaneous effects

---

## 2.2 Setup Stage

**Rule refs:** 2.2.1-2.2.3

The Setup Stage prepares both players to begin the game.

### 2.2.1 Setup Steps (in order)

**1. Randomize Decks** (2.2.1.1)

- Each player shuffles their deck so card order is random
- Players can't look at cards in deck during game

**2. Determine Starting Player** (2.2.1.2)

- Use mutually agreeable random method
- This player will take the first turn

\*\*3. Set Aside Cards for "First to..." (2.2.1.3)

- Any abilities that reference "first to [action]" are tracked
- If multiple players would be first, active player in tiebreaker wins

**4. Draw Opening Hands** (2.2.1.4)

- Each player draws 7 cards
- Starting player skips drawing on their first turn

**5. Alter Starting Hands** (2.2.1.5)

- Each player may choose to alter their starting hand (once per game)
- Take any number of cards from hand, put on bottom of deck (unrevealed)
- Draw until you have 7 cards in hand
- Shuffle deck

**6. Begin Game** (2.2.1.6)

- Starting player begins their turn
- All first-to trackers begin counting

### 2.2.2 Player Communication

**Rule refs:** 2.2.2.1-2.2.2.2

- Players must answer questions about public information truthfully
- Hidden information (hand, facedown cards, deck order) can't be revealed

---

## 2.3 In-Game Stage

**Rule refs:** 2.3.1-2.3.2

The In-Game Stage is when players take turns.

### 2.3.1 General Rules

**Key Concepts:**

- Game continues until a player wins
- Players take turns following turn structure (Section 3)
- Game state checks happen after each effect resolves

### 2.3.2 Ending the Game

**Rule refs:** 2.3.2.1-2.3.2.2

**Win Condition:**

- First player to reach 20+ lore wins immediately (1.11.2)

**Loss Condition:**

- Player who tried to draw from empty deck loses at end of their turn (1.11.3)

---

## 2.4 Cleanup Stage

**Rule refs:** 2.4.1-2.4.2

The Cleanup Stage begins when a player wins the game.

### 2.4.1 Determining the Winner

**Rule refs:** 2.4.1.1-2.4.1.3

**Single Winner:**

- If only one player reaches 20+ lore, that player wins

**Multiple Winners (Tie):**

- If multiple players would win simultaneously, active player wins

### 2.4.2 Ending the Game

**Rule refs:** 2.4.2.1-2.4.2.2

**Game End Actions:**

- Current turn completes (if needed)
- No further turn actions are taken
- All "this turn" effects end
