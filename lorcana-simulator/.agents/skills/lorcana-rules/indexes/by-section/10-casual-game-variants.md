# Section 10: Casual Game Variants

**Comprehensive Rules Version 2.0.1** - Rule References 10.1-10.2

## Overview

This section contains optional rules for casual play. These variants aren't meant to be comprehensive and are for casual play only.

---

## 10.1 General

**Rule refs:** 10.1.1-10.1.2

**Key Concepts:**

- These are optional rules for casual variants
- Not comprehensive list
- Follow normal rules except as defined in variant
- If normal and variant rules conflict, follow variant

**Example:** Normal starting hand is 7. In Pack Rush, starting hand is 5. Use 5 when playing Pack Rush.

---

## 10.2 Pack Rush

**Rule refs:** 10.2.1-10.2.5

Pack Rush is a Limited format where players build decks from booster packs.

### 10.2.1 Setup

**Building Deck:**

- Each player takes two unopened booster packs
- Shuffles them together to play one game
- Booster packs can be from any set
- No minimum deck size
- Deck size = total cards in both packs

**Marketing Cards:**

- Set aside marketing cards before shuffling
- Before drawing opening hands, put 2 marketing cards into inkwell
- Each player starts with 2 cards in inkwell

### 10.2.2 Marketing Cards in Play

- Can't be moved from inkwell to any other zone
- Can't be chosen for effects

### 10.2.3 Starting Hand

- Starting hand size is 5 (instead of 7)

### 10.2.4 Win Condition

- When a player reaches 15 or more lore, they win
- Instead of normal 20 lore

### 10.2.5 Empty Deck

- When player runs out of cards in deck, they don't lose
- Instead, shuffle discard and place facedown as new deck
