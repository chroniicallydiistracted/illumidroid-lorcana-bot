# Section 3: Turn Structure

**Comprehensive Rules Version 2.0.1** - Rule References 3.1-3.4

## Overview

This section describes the three phases that make up each player's turn: Start-of-Turn Phase, Main Phase, and End-of-Turn Phase.

---

## 3.1 Phases

**Rule refs:** 3.1.1-3.1.3

**Key Concepts:**

- Each turn has three phases: Start-of-Turn, Main, End-of-Turn
- Phases happen in order
- Each phase contains specific steps

### Phase Sequence

1. **Start-of-Turn Phase** - Ready cards, gain location lore, draw card
2. **Main Phase** - Take turn actions
3. **End-of-Turn Phase** - Declare end, resolve end-of-turn triggers

---

## 3.2 Start-of-Turn Phase

**Rule refs:** 3.2.1-3.2.4

The first phase of each turn, where players prepare for the turn ahead.

### 3.2.1 Ready Step

**Rule refs:** 3.2.1.1-3.2.1.4

**What Happens:**

1. All exerted cards become ready (3.2.1.1)
2. Effects that last "until start of your next turn" end (3.2.1.2)
3. "During your turn" effects begin applying (3.2.1.3)
4. "At the start of your turn" triggered abilities are added to bag (3.2.1.4)

**Key Terms:**

- **Ready:** Card in upright position, can act
- **Dry:** Character that was in play at start of current turn (can quest, challenge, pay costs)
- **Drying:** Character played after start of turn (can't quest, challenge, or pay costs until next turn)

### 3.2.2 Set Step

**Rule refs:** 3.2.2.1-3.2.2.4

**What Happens:**

1. Resolve all "at start of your turn" triggers from Ready step (3.2.2.1)
2. Gain lore from locations with ◇ value (3.2.2.2)
3. Drying characters become dry (3.2.2.3)
4. New triggers from this step are added to bag (3.2.2.4)

**Key Points:**

- Characters can now quest, challenge, and pay costs
- Location lore is gained during this step

### 3.2.3 Draw Step

**Rule refs:** 3.2.3.1-3.2.3.4

**What Happens:**

1. Resolve all triggers from Set step (3.2.3.1)
2. Draw top card of deck (3.2.3.2)
3. Active player skips this step on first turn of game (3.2.3.3)
4. New triggers from this step are added to bag (3.2.3.4)

**Exception:**

- First turn of game: Starting player skips Draw step

### 3.2.4 End of Start-of-Turn Phase

**Rule refs:** 3.2.4.1-3.2.4.2

**What Happens:**

1. Resolve all triggers from Draw step (3.2.4.1)
2. Move to Main Phase (3.2.4.2)

---

## 3.3 Main Phase

**Rule refs:** 3.3.1-3.3.2

The second phase of each turn, where players take turn actions.

### 3.3.1 General

**Rule refs:** 3.3.1.1-3.3.1.4

**Key Concepts:**

- Active player may take turn actions in any order
- Turn actions are: Ink, Play, Activate Ability, Quest, Challenge, Move
- No limit to number of turn actions (as long as legal)
- Each turn action must complete before next begins

### 3.3.2 Available Turn Actions

During Main Phase, active player may:

| Action                    | Description                                                |
| ------------------------- | ---------------------------------------------------------- |
| **Ink a card**            | Put card from hand into inkwell facedown (4.2)             |
| **Play a card**           | Pay ink cost to play character/action/item/location (4.3)  |
| **Use activated ability** | Pay cost to use ability (4.4)                              |
| **Quest**                 | Exert character to gain lore (4.5)                         |
| **Challenge**             | Exert character to fight opposing character/location (4.6) |
| **Move character**        | Pay move cost to move character to location (4.7)          |

---

## 3.4 End-of-Turn Phase

**Rule refs:** 3.4.1-3.4.5

The third phase of each turn, where the turn ends.

### 3.4.1 Declare End of Turn

**Rule refs:** 3.4.1.1-3.4.1.3

**What Happens:**

1. Active player declares end of turn (3.4.1.1)
2. Can't declare if Reckless character is ready and can challenge (3.4.1.2)
3. Once declared, no further turn actions can be taken (3.4.1.3)

**Reckless Restriction:**

- Can't end turn while ready Reckless character could challenge

### 3.4.2 End of Turn Effects

**Rule refs:** 3.4.2.1-3.4.2.4

**What Happens:**

1. "This turn" effects end (3.4.2.1)
2. "End of your turn" triggered abilities go into bag (3.4.2.2)
3. Resolve all end-of-turn triggers (3.4.2.3)
4. Player to left becomes active player (3.4.2.4)
