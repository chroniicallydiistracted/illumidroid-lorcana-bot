# Lorcana Symbol Registry Audit

Audit whether `docs/port/headless_lorcana_engine_porting_symbol_registry.md` was updated correctly.

Do not implement unless explicitly requested.

Check changed files for new/renamed/removed/meaning-changed symbols.

Each registry entry must include:
- exact symbol/name
- kind
- location
- purpose
- oracle source
- parity notes
- status
- updated date

Output blockers when entries are missing, stale, vague, or not searchable.
